docker安装及docker安装aria2+aria2ui+Filebrowser

安装 docker(如果机器没有的话）

curl -sSL https://get.docker.com/ | sh

service docker start#启动docker

service docker stop#停止docker

service docker restart#重启docker

设置docker开机自启动如下：

systemctl enable docker

一键安装aria2+aria2ui+Filebrowser

docker run -d --name aria2-ui -v /home/down:/data -p 80:80 wahyd4/aria2-ui

Aria2: http://yourip/ui/
FileManger: http://yourip
请使用 admin/admin 进行登录

这样下载的文件都放在 /home/down 这个文件夹里面

文件夹没有写入权限，有兴趣看这篇文章了解 文章链接
不管原理，无脑给予 777

chmod 777 /home/down

如果不想用了可以使用下面的命令删除

docker rm -f aria2-ui

如果想加密 请删除后执行

docker run -d --name ariang
-p 80:80
-e PUID=1000
-e PGID=1000
-e ENABLE_AUTH=true
-e RPC_SECRET=Hello
-e ARIA2_SSL=false
-e ARIA2_USER=user
-e ARIA2_PWD=pwd
-v /home/down:/data
wahyd4/aria2-ui

默认用户名：user
默认密码：pwd
可自行修改

后续补充
根据个人的需求来选择，如果觉得不需要加密 Aria2 的管理界面的，其实用第一种方式就行了。非常简单，一行命令即可。

同时也可以结合 APP ：Transdrone_2.5.15.apk 来使用，这个我放在网盘里，使用它来管理多个下载工具也是比较方便的。
